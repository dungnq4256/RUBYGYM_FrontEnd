## Cài đặt các gói

Dùng lệnh "npm install" để cài đặt các gói cần thiết.


## Chạy chương trình

Dùng lệnh "npm start" để chạy chương trình ở port 4000.


## Xây dựng chương trình

Dùng lệnh "npm run build" để xây dựng chương trình.
